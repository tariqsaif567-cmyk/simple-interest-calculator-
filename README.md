# simple-interest-calculator-

This is a simple Bash script to calculate Simple Interest.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## How to Run

1. Give execute permission:
   chmod +x simple-interest.sh

2. Run the script:
   ./simple-interest.sh

## Example
Input:
- Principal = 1000
- Rate = 5
- Time = 2

Output:
- Simple Interest = 100
